# 🔐 Cybersecurity Labs & Notes

Este repositório reúne meus **laboratórios práticos, anotações técnicas e experimentos em cibersegurança** realizados durante meus estudos na área.

O objetivo é documentar minha evolução prática em **segurança da informação, redes e ethical hacking**, utilizando ambientes de laboratório como **Kali Linux, Windows Server e ferramentas de análise de rede**.

---

## 🎯 Objetivo

Construir uma base sólida em **Cybersecurity**, com foco inicial em:

- Network Security  
- Blue Team (detecção e análise)  
- Ferramentas de enumeração e análise de rede  
- Sistemas Linux aplicados à segurança  

No futuro, pretendo expandir também para **Red Team, Pentest e análise de vulnerabilidades**.

---

## 🧠 Metodologia de Estudo

O aprendizado é organizado através de:

- 📚 **Resumos técnicos** das ferramentas e conceitos
- 🧪 **Laboratórios práticos** realizados em ambiente controlado
- 📝 **Documentação em Markdown** para facilitar consulta
- 🗂 **Organização por temas de segurança**
- 🔧 **Versionamento no GitHub**

---

## 🛠 Tecnologias e Ferramentas

Algumas das ferramentas e tecnologias utilizadas nos laboratórios:

- Kali Linux
- Windows Server
- Nmap
- Wireshark
- DNS Tools
- Bash / Linux CLI
- Virtualização (VirtualBox)
